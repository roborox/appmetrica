# @roborox/appmetrica

Yandex AppMetrica wrap module

## Installation

```sh
npm install @roborox/appmetrica
```

## Usage

```js
import Appmetrica from "@roborox/appmetrica";

// ...

const result = await Appmetrica.multiply(3, 7);
```
